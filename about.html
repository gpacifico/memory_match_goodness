About the Game:

Steven Matchiverse is a memory matching game of skill and concentration, pitting you against forgetfulness. You must find the matching pairs of Steven Universe characters. Once you match them all, you win! The Crystal Gems will be proud of you!


The list of skills needed for the creation of this game are:

  • HTML5
  • CSS3
  • JavaScript
  • Knowledge of Steven Universe
